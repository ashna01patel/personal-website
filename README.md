# Personal Website

A macOS-style interactive portfolio built with React and Vite. The desktop mimics the look and feel of macOS — draggable windows, a dock, and app icons — with each "app" serving as a different section of the site.

## Apps

- **Terminal** — intro and about me
- **Experience** — work history and background
- **Messages** — contact / reach out
- **Safari** — projects
- **Photos** — artwork gallery with lightbox view
- **Notes** — personal thoughts and writing

## Tech

- React + Vite
- Framer Motion for window animations
- CSS with macOS-inspired dark UI

## Running locally

```bash
npm install
npm run dev
```
